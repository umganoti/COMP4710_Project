COMP4710 Project - Direct Candidate Generation Readme

Files needed: 
dcgtable.txt or a text file with exact format as dcgtable.txt

Text format:
First line represents the number of transactions. 
All subsequent lines start with the transaction number, followed by 2 tabs, number of items, 
followed by 2 tabs, single spaced items listed in order, followed by 3 tabs 
and the quantity of corresponding items.

How to run the program:
From the command line, compile files and enter the following: program_name filename.txt minshare_value (float)
ex: ./a.out dcgtable.txt 0.3
